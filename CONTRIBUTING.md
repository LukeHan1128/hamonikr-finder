
하모니카 개발팀은 bitbucket, github 2가지 저장소를 사용합니다.

릴리즈 하기 전에는 공개되지 않은 저장소에서 개발을 진행하고 릴리즈를 결정한 이후에는 깃헙을 사용합니다.

깃헙에 공개된 프로젝트는 누구나 참여 가능하며, 하모니카 OS 코어 개발팀은 팀 리더의 승인이 필요합니다.
